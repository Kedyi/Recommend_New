## Run Locally

Clone the project

```bash
  git clone https://github.com/Kedyi/recommend-new.git
```

Go to the project directory

```bash
  cd recommend_new
```

### Install dependencies from requirements.txt
## Start the server

```bash
  Flask --app app  run
```
